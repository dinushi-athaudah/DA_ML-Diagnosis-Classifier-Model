# ME/CFS vs. Depression ML Classification

This repository contains a machine learning pipeline designed to differentiate between ME/CFS and Depression using various classification algorithms.

## Project Structure
- `ME_CFS Vs Depression_ML Classification.ipynb`: Main Jupyter Notebook.
- `Data/`: Folder containing the `me_cfs_vs_depression_dataset.csv`.
- `requirements.txt`: List of Python dependencies.

## Setup Instructions
1. Clone the repository.
2. Install requirements:
   ```bash
   pip install -r requirements.txt
3. Run the notebook using Jupyter:
   ```bash
   jupyter notebook
